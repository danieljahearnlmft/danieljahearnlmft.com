How to install this slim signature

  1. Click anywhere inside the dashed box below
  2. Press Cmd+A to select all, then Cmd+C to copy
  3. Open Gmail → Settings (gear) → See all settings → General → scroll to Signature
  4. Click Create new → name it "Daily" (or "Slim") → paste with Cmd+V
  5. Under Signature defaults set this slim one as default for both new emails and replies/forwards
  6. Save
